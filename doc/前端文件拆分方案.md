### tabs/ 目录

每个文件对应一个 `tab-content` 区块：

| 文件 | 对应 ID | 内容 |
|------|---------|------|
| dashboard.html | tab-dashboard | 仪表盘统计+图表 |
| employees.html | tab-employees | 员工表单+表格 |
| employees-attendance.html | tab-employees-attendance | 头像上传+人脸考勤 |
| attendance.html | tab-attendance | 考勤表单+表格 |
| attendance-settings.html | tab-attendance-settings | 考勤规则设置 |
| salary.html | tab-salary | 薪资表单+表格 |
| org.html | tab-org | 部门/岗位/职级配置 |
| permissions.html | tab-permissions | 权限表单+表格 |
| import-export.html | tab-import | 导入导出面板 |

### 加载机制

```javascript
// tabs.js 中动态加载
async function loadTabHTML(tabName) {
  const container = $(`tab-${tabName}`);
  if (!container) return;
  const html = await fetch(`tabs/${tabName}.html`).then(r => r.text());
  container.innerHTML = html;
}
```
