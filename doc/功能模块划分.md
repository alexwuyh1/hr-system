# HR系统功能模块划分

## 模块概览

| 模块 | 包含子模块 | API路径 |
|------|-----------|---------|
| 人事考勤 | 员工、组织、考勤、薪资 | `/api/employees`, `/api/departments`, `/api/attendance`, `/api/salaries` |
| 报表展示 | 仪表盘、报表 | `/api/dashboard`, `/api/reports` |
| 系统管理 | 认证、权限、角色、导入导出 | `/api/auth`, `/api/permissions`, `/api/data` |

## 详细功能

### 人事考勤模块

| 子模块 | 功能 | 接口 |
|--------|------|------|
| 员工管理 | CRUD、入离职、头像、人脸验证 | `/api/employees`, `/api/face` |
| 组织管理 | 部门树、岗位、职级 | `/api/departments`, `/api/positions`, `/api/grades` |
| 考勤管理 | 打卡记录、考勤规则、排班、请假、加班 | `/api/attendance`, `/api/attendance-rules`, `/api/shifts`, `/api/leaves`, `/api/overtime` |
| 薪资管理 | 薪资CRUD | `/api/salaries` |

### 报表展示模块

| 子模块 | 功能 | 接口 |
|--------|------|------|
| 仪表盘 | 数据概览 | `/api/dashboard` |
| 报表 | 汇总报表 | `/api/reports` |
| 数据聚合 | 初始化计算 | `/api/init` |

### 系统管理模块

| 子模块 | 功能 | 接口 |
|--------|------|------|
| 认证 | 登录、注册、JWT | `/api/auth` |
| 权限管理 | 权限CRUD | `/api/permissions` |
| 角色管理 | 角色列表 | `/api/roles` |
| 用户管理 | 用户列表、角色分配 | `/api/users` |
| 数据工具 | CSV/Excel导入导出 | `/api/data` |

## 模块依赖

```
系统管理（认证）→ 所有模块（前置）
人事考勤 → 报表展示（数据源）
系统管理（导入导出）→ 人事考勤（员工/考勤/薪资数据）
```
