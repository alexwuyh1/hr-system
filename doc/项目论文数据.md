### 人脸识别底层技术
本项目未使用 MTCNN、FaceNet 等深度学习方案，而是采用轻量级传统图像处理方案：

| 层面 | 说明 |
|------|------|
| **图像采集** | 浏览器端上传照片，后端接收为字节流 |
| **预处理** | OpenCV 灰度化 (BGR→GRAY) + 双线性插值缩放到 200×200 |
| **特征表示** | 将灰度图像展平为一维像素向量 (40,000 维) |
| **相似度计算** | 余弦相似度: cos(θ) = A·B / (‖A‖ × ‖B‖) |
| **判定阈值** | 0.75 (75% 相似度以上判定为同一人) |
| **MVP 定位** | 适用于光线均匀、正面拍摄的办公场景，不依赖 GPU 或深度学习模型 |

**与深度学习方案对比**:
| 方案 | 精度 | 部署成本 | 适用场景 |
|------|------|----------|----------|
| FaceNet + MTCNN | 高 (99%+) | 需 GPU/模型文件 | 大规模安防 |
| OpenCV 余弦相似度 | 中 (75%+) | 零额外依赖 | 轻量级考勤 |

#### 3.3.2 人脸签到/签退

**流程图**:
```
选择员工 → 上传照片 → 人脸控制器
    │
    ├── 1. 查询员工信息（验证在职状态）
    ├── 2. 读取员工头像文件
    ├── 3. OpenCV 预处理（灰度化 + 缩放）
    ├── 4. 计算余弦相似度
    ├── 5. 对比阈值 (75%) 返回结果
    └── 6. 匹配成功 → 创建考勤记录
```

**核心代码**:
```java
// 1. 验证员工状态必须为在职
if (!"在职".equals(employee.getStatus())) {
    throw new EmployeeNotActiveException(employeeId);
}
// 2. 调用 OpenCV 计算人脸相似度
FaceResult result = faceRecognitionService.verify(probeImage, referenceImage);
// 3. 相似度低于阈值则拒绝签到
if (result.similarity() < FACE_SIMILARITY_THRESHOLD) {
    throw new FaceVerificationFailedException(result.similarity(), threshold);
}
// 4. 创建考勤记录并保存
attendance.setCheckIn(LocalTime.now());
return attendanceRepository.save(attendance);
```
